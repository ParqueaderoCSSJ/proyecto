HEADER Y BARRA DE NAVEGACION

En nuestro header encontramos tres logos, dos de ellos cumplen la función de redireccionarnos a nuestra pagina principal que en este caso es disponibilidad, y a todo el costado derecho esta el logo de logout o cerrar sesión, que nos lleva hacia la pagina de login.

La barra de navegación es la fila que esta pegada a nuestra margen izquierda donde encontramos el nombre de cada subpagina que abarcamos en este proyecto, como disponibilidad, facturas, mensualidad, reservas, registro y ayuda.

Disponibilidad

En esta primera pagina llamada disponibilidad encontramos tres graficas, para las celdas generales, para las celdas de reservas y para las celdas de mensualidad, tanto para carros como para motos. Todo esto para llevar un control de celdas disponibles y ocupadas del establecimiento.

Facturas

en facturas podemos encontrar la funcionalidad donde el controlador del sistema va ingresar los datos necesarios del cliente y el administrador podra llevar de manera organizada y a tiempo real el conteo de clientes que maneja el sistema.

Mensualidad

Mensualidad es muy similar a la anterior, sigue siendo el mismo estilo de manejo, pero esta ahora guardara datos de un cliente que desea contratar una mensualidad en el parqueadero

Reservas

En reservas se sigue conservando el estilo de las anteriores, pero ahora desde esta pagina vaamos poder ver los datos de los clientes que han realizado una reserva para hacer uso del parqueadero.

Registro

Registro se dedica a cuando hay un nuevo empleado, crearle y asignarle un usuario unico para que controle el sistema y asi mantener datos exactos de las personas que controlan y realizan cambios en las distintas paginas del parqueadero.
